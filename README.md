A lightweight Python toolkit for converting images into transparent PNGs, favicons, and multi-size logo assets.  
Built with Pillow and optimized for fast, automated image processing.

---

## Features

- Remove dark backgrounds and export **transparent PNGs**
- Generate **ICO favicons** and **48Ã—48 PNG icons**
- Convert logos into multiple sizes for:
  - Websites
  - Mobile apps
  - Retina displays
  - Social media profiles
- Clean, simple command line usage
- No external dependencies except Pillow
